Impor Data Peserta

Unggah berkas CSV untuk memperbarui seluruh data hasil lomba secara massal.

📥 Unduh Template Excel

*Setelah mengisi data di Excel, simpan kembali (Save As) ke format CSV (Comma delimited) sebelum diunggah.

💡 Informasi Kolom CSV:

- Kolom wajib: bib (atau no_bib), name, category (jarak), gender, gun_time, net_time.
- Kolom tambahan (opsional): checkpoint (waktu CP), rank_overall, rank_category_gender, age_group, city, photo_url.
- Format data gender yang dikenali: L / Putra untuk laki-laki, P / Putri untuk perempuan.
